# Citation

Bryan, George H., and Richard Rotunno, 2009b: *The Maximum Intensity of Tropical Cyclones in Axisymmetric Numerical Model Simulations*. *Monthly Weather Review*, 137, 1770-1789. DOI: [10.1175/2009MWR2709.1](https://doi.org/10.1175/2009MWR2709.1).

Accessed text sources: [AMS journal page](https://journals.ametsoc.org/view/journals/mwre/137/6/2009mwr2709.1.xml) and [Montgomery and Smith (2017) review synthesis](montgomery-smith-2017-fluid-dynamics-of-tropical-cyclones.md).

Peer-review status: Peer-reviewed primary numerical-model research article.

# Research Question

Bryan and Rotunno (2009b) ask how maximum achievable intensity behaves across axisymmetric numerical simulations and how that behavior compares with analytical PI expectations.

# Data And Methods

- Study type: Idealized axisymmetric numerical simulation study.
- Experiment logic: Vary core modeling assumptions and environmental/parameter settings to map simulated maximum-intensity outcomes.
- Diagnostics: Maximum wind and pressure behavior in relation to PI-style theoretical expectations.

# Findings

- Bryan and Rotunno (2009b) provide simulation-based constraints on how maximum intensity behaves in idealized axisymmetric storms.
- The paper is a core companion to Bryan and Rotunno (2009a), together forming a key testing branch between analytical PI theory and explicit model behavior.
- Inference from [Montgomery and Smith (2017)](montgomery-smith-2017-fluid-dynamics-of-tropical-cyclones.md): this axisymmetric testing branch supports treating PI as useful but assumption-sensitive rather than as a closed theory.

# Limitations

- Current page is a constrained first pass using bibliographic metadata and review-level synthesis; detailed sensitivity tables and quantitative thresholds require full-text extraction.
- Axisymmetric modeling excludes key 3-D effects that can alter intensity pathways in real storms.
- Comparison to observed storms is indirect.

# tc-wiki

`tc-wiki` is a persistent markdown wiki of peer-reviewed tropical cyclone research. It is organized as an evolving knowledge base of papers, topics, methods, and syntheses.

Start with the [Wiki](wiki/index.md). Recent repository updates are tracked in [wiki/log.md](wiki/log.md).

The repository also includes the [skill file](.agents/skills/tc-wiki/SKILL.md) that defines the workflow and conventions to maintain the wiki with an AI agent.

Inspired by Andrej Karpathy's [llm-wiki.md](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f).
